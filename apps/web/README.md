# apps/web

Separate `Nuxt 3` frontend for the API-first migration.

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Runtime

- default API base: `http://localhost/api/v1`
- override with `NUXT_PUBLIC_API_BASE`

## Current pages

- `/`
- `/catalog`
- `/catalog/category/:slug`
- `/basket`
- `/checkout`
- `/login`
- `/profile`
- `/orders`
